# Data-Cleaning-Sales-Analysis-Project-Vee-Inc.-Junior-Data-Professional-Task

# Project Overview
This project was completed as part of my role as a Data Professional at Vee Inc. The objective was to clean and prepare a client’s dataset, resolve inconsistencies, and generate key business metrics based on enquiries from the Team Lead. All tasks were completed using Microsoft Excel, applying a range of data cleaning and analytical techniques.

# Project Objectives
The Team Lead required the following tasks:
	Clean the dataset and correct inconsistencies
	Separate customer names into First Name and Last Name
	Calculate overall revenue, cost, and profit in USD
	Compute discounts for each sale at 5%, 7%, and 10% levels

# Data Cleaning Techniques Used
To prepare the dataset for analysis, I applied the following Excel tools and functions:
	Text to Columns – for splitting names
	TRIM – to remove extra spaces
	SUBSTITUTE – to correct inconsistent characters
	Remove Duplicates – to eliminate repeated entries
	Flash Fill – for pattern based transformations
	PROPER, UPPER, LOWER – for text formatting
	Find & Replace – for quick corrections
	Paste Special – to convert formulas to values
These steps ensured the dataset was clean, standardized, and ready for accurate calculations.

# Analysis Performed
✔️ Name Separation
Customer names were split into First Name and Last Name to improve clarity and enable better sorting, filtering, and reporting.
✔️ Revenue, Cost & Profit Calculation
Using the cleaned dataset, I calculated:
	Total Revenue
	Total Cost
	Profit (Revenue − Cost)
These metrics help assess overall business performance.
✔️ Discount Analysis
Discount amounts were computed for each sale using the required discount levels:
	5% Discount
	7% Discount
	10% Discount
This allows comparison of how different discount strategies impact sales.

# Key Formulas Used
	Revenue 
Revenue = Unit Price × Quantity
	Cost 
Cost = Cost Price × Quantity  
	Profit 
Profit = Revenue − Cost
	Discount Amount 
	Discount = Revenue × Discount Rate 

# Project Outcome
	A fully cleaned and standardized dataset
	Clear separation of customer names
	Accurate revenue, cost, and profit calculations
	Discount analysis across multiple discount levels
	Demonstration of strong Excel based data cleaning and analytical skills
